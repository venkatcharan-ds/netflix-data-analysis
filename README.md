# Netflix Data Analysis

## Overview
This project analyzes Netflix Movies and TV Shows data using Python and Data Analytics techniques.

The goal of this project is to explore:
- Content trends
- Popular genres
- Ratings distribution
- Movies vs TV Shows
- Release year analysis
